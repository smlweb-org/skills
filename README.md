# S—ML Web · skills

AI-agent skills for building and aligning web pages to the [S—ML Web standard](https://smlweb.org) — Sparse · Medium · Luxe delivery profiles.

The premise: most new pages are increasingly generated by AI tooling. If the tooling knows the standard, pages are **born conforming** — the author never needs to learn what a budget or hydration is. These skills are that knowledge, packaged.

## Skills

### s-ml-align

Teaches a capable AI agent (Claude-class) to build new pages Sparse-first and to align existing pages to the profiles — including the budgets table, both workflows, discovery/negotiation snippets, and the bundled conformance auditor.

- [`s-ml-align/`](s-ml-align/) — the skill as a readable folder (SKILL.md + normative spec reference + auditor script). Adapt freely to other agent ecosystems.
- [`s-ml-align.skill`](s-ml-align.skill) — the same skill packaged for one-click install in Claude. (A `.skill` file is a zip of the skill folder.)

## Contributing

New skills and adapters welcome — framework scaffolds, generation-platform plugins, ports to other agent formats. Open an issue describing the target ecosystem first. Skill behavior must track the normative spec in [smlweb-org/spec](https://github.com/smlweb-org/spec); when the spec changes, skills follow.
